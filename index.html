CSS Grid Layout

CSS Grid layout contiene funciones de diseño dirigidas a los desarrolladores de aplicaciones web.
El CSS grid se puede utilizar para lograr muchos diseños diferentes. También se destaca por permitir
dividir una página en áreas o regiones principales, por definir la relación en términos de tamaño,
posición y capas entre partes de un control construido a partir de primitivas HTML.
Al igual que las tablas, el grid layout permite a un autor alinear elementos en columnas y filas.
Sin embargo, con CSS grid son posibles muchos más diseños y de forma más sencilla que con las tablas.
Por ejemplo, los elementos secundarios de un contenedor de cuadrícula podrían posicionarse para que
se solapen y se superpongan, de forma similar a los elementos posicionados en CSS.

Propiedades:

  • grid-template-columns: define el nombre de las líneas y las funciones de tamaño de línea de grid columns.
  • grid-template-rows: define el nombre de las líneas y las funciones de tamaño de línea de grid rows.
  • grid-template-areas: especifica nombres para cada una de las grid areas.
  • grid-template: CSS property is a shorthand property for defining grid columns, grid rows, and grid areas.
  • grid-auto-columns: especifíca el tamaño de una columna de cuadrícula creada implícitamente
  • grid-auto-rows: de CSS especifíca el tamaño de una nueva fila creada de forma implícita.
  • grid-auto-flow: CSS como fluyen los elementos de forma automatica en el grid
  • grid: es un shorthand que permite definir todas las propiedades grid explícitas, implícitas y relativas en una sola declaración.
  • grid-row-start: CSS property specifies a grid item's start position within the grid row
  • grid-column-start: CSS property specifies a grid item's start position within the grid column
  • grid-row-end: CSS property specifies a grid item's end position within the grid row
  • grid-column-end: CSS property specifies a grid item's end position within the grid column
  • grid-row: CSS shorthand property specifies a grid item's size and location within a grid row
  • grid-column: CSS shorthand property specifies a grid item's size and location within a grid column
  • grid-area: CSS shorthand property specifies a grid item's size and location within a grid
  • row-gap: CSS property sets the size of the gap (gutter) between an element's rows.
  • grid-column-gap: especifica el gutter entre los grid columns.
  • rid-gap: es una propiedad abreviada shorthand para grid-row-gap y grid-column-gap que especifica los canales entre las filas y las columnas de la cuadrícula.
